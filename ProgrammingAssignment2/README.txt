Compile/run:

start with PINGServer.py
in command line, type python PINGServer.py <port number>
replace <port number> with an actual port
example:    python PINGServer.py 22
PINGServer.py should now be waiting for client input

then run PINGClient.py
in command line on a different terminal, type python PINGClient.py <host name> <port number>
replace <host name> with a host and <port number> with an actual port
example:    python PINGClient.py localhost 22
